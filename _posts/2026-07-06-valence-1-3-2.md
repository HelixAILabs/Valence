A fix for how Valence updates itself. New here? Start with [Valence 1.3.0]({{ '/2026/07/06/valence-1-3-0.html' | relative_url }}) &mdash; the slim installer debut and the reimagined first run.

## Fixes

- Installing an update no longer shows a false "download failed" message. The update was always installing correctly underneath, but a threading timing issue surfaced a `CoreWebView2` error and left the window looking stuck. The installer launch now runs on the UI thread, so updates from here on are clean and quiet.
