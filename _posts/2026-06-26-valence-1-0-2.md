Valence 1.0.2 is out — a round of interface polish, a friendlier first run, and a batch of local-AI fixes.

## What's new
- **AI Providers, reimagined** — local, cloud, and enterprise in one calm, unified list.
- **A friendlier first run** — the setup wizard tells you, in plain language, how well a local model will run on your hardware before you download it.
- **See it think** — a live "Thinking…" indicator while a local model reasons through a hard question.
- **Check for updates, anytime** — a button is back in Settings → About, with an optional "check on startup" toggle (off by default).

## Fixes
- **Ollama** now lists the models you've actually installed — and tells you clearly when Ollama isn't running — instead of a generic placeholder list.
- **Local models** give clear guidance when your NVIDIA graphics driver is too old to run them, so you can update instead of hitting a confusing error.
- Removed a stray "add an API key" prompt that appeared for local models, which don't need one.
