Getting the latest Valence is now clearer and tidier.

## What's new

- **You'll know when an update is ready.** With "check on startup" turned on, Valence now tells you the moment a new version is available &mdash; the prompt actually appears on launch instead of staying silent.
- **Watch it arrive.** When you install an update, a little robot trots off to the web and carries the new version back, with live progress.
- **Tidier downloads.** Valence now cleans up old installer files on startup, so updates don't quietly leave gigabytes of leftovers on your disk.
